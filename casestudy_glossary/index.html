PROJECT – E-Learning Glossary
CLIENT – Academy of Art University
ROLE - User Interface Design

The Brief
Each online class has a glossary of new terms, which are introduced within the class content. The aim is to create a glossary tool, which is more consistent with the learning needs of the students than the one currently being used.

Current Glossay
New Term Within Class Page

The Approach
The team worked closely with a student user group to define the educational needs.

Download UI Deliverables
User Scenarios (pdf) | Features List (pdf)

Features were included to allow the students to use the glossary as a study aid; this included creating study cards from definitions and a pronounciation feature to help ESL (english as a second language) students.

Download UI Deliverables
Storyboard1 (pdf) | Storyboard2 (pdf) | Storyboard3 (pdf)

The Results
All new terms are linked from the modules in which they appear. Students can view the terms by class, by module or by bookmaking them and creating a study group. The contents of the glossary can be emailed or printed out as needed. Creation of a different glossary for each individual class will be streamlined, with the designers only needing to fill out an html form.
